== Description ==
**Better Alipay Payment Gateway for WooCommerce** is a payment gateway that allows your WooCommerce powered store / e-commerce website to accept credit card and PayPal payment via Alipay.

== Installation ==

Navigate to your WordPress "Plugins" page, inside of your WordPress dashboard, and follow these instructions:

1. In the search field enter **Better Alipay Payment Gateway for WooCommerce**. Click "Search Plugins", or hit Enter.
1. Select **Better Alipay Payment Gateway for WooCommerce** and click either "Details" or "Install Now".
1. Once installed, click "Activate".
